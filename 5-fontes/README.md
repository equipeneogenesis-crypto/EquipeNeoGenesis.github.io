# 5-fontes/ — Fontes Personalizadas

Fonte **Supercharge** (vários pesos e estilos). Instalada localmente.

## Arquivos

| Arquivo | Estilo |
|---------|--------|
| `supercharge.ttf` | Regular |
| `superchargestraight.ttf` | Straight (sem inclinação, usado como bold/900) |

## Uso

As fontes Google Fonts (Orbitron, Exo 2, Playfair Display, Inter) são carregadas via CDN. A Supercharge está disponível localmente caso queira usar em títulos ou elementos especiais.

Para usar, declare no CSS:

```css
@font-face {
  font-family: 'Supercharge';
  src: url('../5-fontes/supercharge.ttf');
}
```
