# 4-recursos/imagens/ — Imagens do Site

## Estrutura

```
imagens/
├── membros/          → Fotos dos integrantes (referenciadas em 1-dados/membros.js)
├── robos/            → Fotos dos robôs (referenciadas em 1-dados/robos.js)
├── hero-bg.jpeg      → Background do hero na landing page
├── logo.jpeg         → Logo da equipe (usado na navbar e footer)
├── robo-bg.jpeg      → Imagem de fundo/generic para robôs
├── slide-1.jpeg      → Slide 1 do carrossel (index.html)
├── slide-2.jpeg      → Slide 2 do carrossel
├── slide-3.jpeg      → Slide 3 do carrossel
├── renet.png         → Logo Renet (patrocinador)
├── whatsapp-dev.jpeg → Imagem seção desenvolvimento
└── WhatsApp*.jpeg    → Imagens diversas
```

## Convenções

- Imagens quadradas para cards: ~400×400px, formato jpg
- Imagens de slide: ~1200×600px, formato jpeg
- Logos: PNG com fundo transparente
- Nomes sem acentos ou espaços (use hífen)
